# Desafio ContaBanco: Crie um Sistema Bancário em Java!

## Objetivo:

Construir um sistema de contas bancárias em Java usando o terminal.

## Desafio:

- Crie contas com:
  - Número da conta
  - Nome do cliente
  - Agência
  - Saldo inicial
- Valide os dados de entrada.
- Exiba mensagem final com os dados da conta.
